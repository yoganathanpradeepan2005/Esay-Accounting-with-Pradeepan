# Easy Accounting with Pradeepan

A modern educational website for **Easy Accounting with Pradeepan**, designed to help AAT and Accounting students learn accounting in a simple, clear, and engaging way.

## 📚 Features

- Responsive modern website
- Course information
- Theory Classes
- Paper Classes
- Model Papers
- Student-friendly interface
- Contact section
- Mobile-friendly design
- Smooth animations
- Fast loading

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yoganathanpradeepan2005/Esay-Accounting-with-Pradeepan.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.

No additional installation is required.

## 🎯 Purpose

This project was created to provide students with an easy way to access accounting learning resources, class information, and educational content.
